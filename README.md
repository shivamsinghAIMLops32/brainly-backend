# BRAINLY PROJECT USING MERN STACK AND TYPESCRIPT

```bash
 npm init -y

 npm i -D typescript

 npx tsc --init
```

# Run script

```bash
npm run dev
```

## Basic change to make in package.json

````js
   "build": "tsc -b",
   "start": "node ./dist/index.js",
   "dev": "npm run build && npm run start"
   ```
````
